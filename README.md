# 🚀 DSBDA Lab Practicals

This repository contains all practical assignments and implementations for the **Data Science and Big Data Analytics (DSBDA)** subject of **SPPU TE Computer Engineering**.

The repository includes Jupyter Notebook practicals, data analysis programs, machine learning algorithms, visualization techniques, and assignment outputs developed using Python libraries commonly used in Data Science.

---

## 📚 Practicals Included

- Practical 1 – Data Preprocessing
- Practical 2 – Data Cleaning & Handling Missing Values
- Practical 3 – Data Visualization
- Practical 4 – Statistical Analysis
- Practical 5 – Linear Regression
- Practical 6 – Logistic Regression
- Practical 7 – Decision Tree Classification
- Practical 8 – Naive Bayes Classification
- Practical 9 – K-Means Clustering
- Practical 10 – Text Analytics & Data Mining

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
